Apply Request Flow

What happens when the user clicks "Apply all" — snapshot, write, log, return

Snapshot rules

  • • Only touched paths copied — not full .claude/
  • • Tombstones for absent files (so install→revert deletes correctly)
  • • Snapshot dir keyed by change_id, never reused
  • • See app/services/snapshot.py:snapshot_path

history.jsonl format

  • • one JSON object per line
  • • fields: change_id, ts, kind (apply|revert), summary, ops[]
  • • kind=revert entries reference the original change_id
  • • Reverts are themselves revertible

Op kinds

  • • install_skill / remove_skill
  • • install_statusline
  • • patch_settings (deep-merge)
  • • scaffold_docs (writes prompt file)
